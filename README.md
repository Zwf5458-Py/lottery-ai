## 📖 项目简介

本项目是一款专为澳门六合彩（Macau Mark Six）设计的全栈轻量级分析工具。系统不仅深度聚焦于**特码（Special Number）**的维度打法研究，更史无前例地集成了 **Google Gemini 2.5 Pro AI 大模型**，通过抓取海量历史真实开奖数据与 AI 推理深度结合，为用户提供科学与智能并重的沉浸式外围下注参考。

> [!IMPORTANT]
> **免责声明**：本程序的所有统计数据、AI 推荐、趋势分析及模拟结果仅供统计学研究参考。彩票开奖为独立随机事件，任何算法与 AI 模型均无法打破纯粹的随机性预测未来结果。请理性对待，严禁用于任何形式的非法博彩。

## ✨ 核心功能特性

### 1. 🤖 深度集成 Google Gemini AI
- **旗舰 AI 推理引擎**：底座接入 `gemini-2.5-pro` 旗舰大语言模型，赋予数据分析以逻辑思考能力。
- **自定义维度下探**：用户可任意组合 大小走势 / 单双走势 / 冷热频率 / 尾数 / 生肖，AI 会动态生成分析 Prompt 并输出精准解读。
- **多模态视觉发现**：生成走势折线点阵图，交由大模型视觉神经进行分析，通过观察“M头”、“双底”、“三重对称”等几何技术形态规律融入最终结论。

### 2. 🧠 纯正深度学习与时序信号 (Machine Learning)
- **MLPClassifier 神经网络**：彻底摒弃简单的伪算法打分，系统底层现已集成 `scikit-learn`。读取过去近 500 期的全量特码真实数据，提取 `(近50期频率, 近10期频率, 实时遗漏期数)` 作为特征列，动态训练多层感知机 (MLP) 神经网络。
- **动态权重推演**：使用 `predict_proba` 输出最新走势在真实数据背后的非线性概率分布，精准研判“强势突破”、“震荡上行”或“深度回调”。
- **贝叶斯极值反弹**：结合生肖的真实历史最大遗漏值，运用贝叶斯推断公式，当某一生肖长期未出逼近历史极值时，给予呈指数级爆增的后验权重（俗称“捉冷底”）。

### 3. 🧮 均值回归与博弈反跳机制
- **长龙断龙阻击**：当连续出现长龙（连大/连偶等）时，传统内核系统会根据连数急剧拉升“反转断龙”概率。
- **马尔可夫链跃迁**：根据数万条历史开奖数据构建生肖与波色的状态转移矩阵。若上一期开出“虎”，系统能精准查表推演出下一期最容易跃迁（转移）的生肖。
- **高频单跳捕捉机制**：实时探测走势中是不是在以"大-小-大-小"或"单-双-单-双"的形态"单跳"。当交替次数 ≥3 次时，系统底层内核赋予极其激进的“防跳（连庄断交替）”加权判定，全方位模拟实战心态。

### 4. 🔥 特化与剥离的数据逻辑
- **完美切割分离**：正码与特码预测解耦。不仅单双/大小仅应用于专门用于特码，连【冷热频率】的统计与读取也已被全部改造为“纯特码维度”，不掺杂正码热度干扰。正码将依据大盘随机或纯冷热体系走，而特码具备独立的复合加权逻辑。

### 5. 🛡️ 生产级高可用与容灾保障
- **多数据源自动容灾 (Fallback Sources)**：系统不再单点依赖单一官网 API。建立主备镜像链路 (`macaujc.com` -> `history.macaumarksix.com`)。当主源被拦截或宕机时，数据守卫瞬间无缝切换至备用源，确保数据永远最新。
- **内存级极速缓存 (Flask-Caching)**：由于彩票数据一天仅变动一次，所有沉重的百万次矩阵计算结果均被缓存在内存中（直到新数据拉取时触发失效），使 API 响应达到毫秒级。
- **API 接口防刷限流 (Rate Limiting)**：对于消耗高额算力和 Token 的 AI 推演接口，强制套用防刷桶机制（如 `10次/分钟`），保障底层大模型稳定安全。
- **全链路日志追踪**：舍弃脆弱的 print，部署了自动轮转的 `RotatingFileHandler` 工业级日志，任何异常波动均可追溯。

## 🛠️ 技术架构

- **后端底座**: Python 3.10 + Flask (轻量、极速)
- **前端渲染**: HTML5 + Vanilla CSS + Chart.js
- **大模型核心**: Google GenAI SDK (Gemini-2.5-Pro)
- **数据流**: Pandas 高性能提纯 + SQLite3 单文件持久化
- **安全保障**: `secrets.SystemRandom` 密码学级随机内核保障底线公平性

## 🚀 快速启动

### 1. 安装环境与依赖
请确保你的环境预装 Python 3 并在根目录下执行：
```bash
pip install -r requirements.txt
```

### 2. 数据初始化（如果无库或需补全）
运行数据同步脚本快速抓取真实记录（包含但不限于 2021-2026年数据）：
```bash
python data/fetch_real_data.py
```

### 3. 一键挂载启动
```bash
python app.py
```
终端将会提示在 `http://127.0.0.1:5000` 提供服务。直接在任意现代浏览器中打开即可开始进行 AI 深度推演。
